O que são as Medidas Relativas?

As medidas relativas no CSS são unidades de tamanho que se ajustam de acordo com o contexto do elemento ao qual estão aplicadas. Diferentemente das medidas absolutas (como px, cm ou in), as medidas relativas são mais flexíveis, permitindo designs responsivos que se adaptam a diferentes dispositivos e tamanhos de tela.

Principais Medidas Relativas no CSS

  1. em

    • Baseia-se no tamanho da fonte do elemento pai.
    • Exemplo: Se a fonte do elemento pai é 16x, 2em será igual a 32px.
    • Geralmente usando para espaçamentos, tamanhos de fonte e margens.

  2. rem (root em)

    • Baseia-se no tamanho da fonte do elemento raiz(<html>).
    • xemplo: se o tamanho da fonte no elemento <html> é 16px, 2rem será igual a 32px
    • Mais previsível que em em layouts complexos.

  3. %

    • Baseia-se no tamanho do elemento pai (ou, no caso de largura, no tamanho do contêiner).
    • Exemploo: widht: 50% define a largura do elemento como metade do contêiner pai.
    • Usado para largura, altura e outros dimensionamentos.

  4. vh e vw (viewport height e viewport widht)

    • Relacionam-se ao tamanho da janela de visualização:
    • 1vh = 1% da altura da viewport.
    • 1vw1% da largura da viewport.
    • Usado para dedsigns responsivos que dependem do tamanho da tela.

  5. vmin e vmax

    • Baseiam-se no menor(vmin) ou maior (vmax) valor entre a altura (vh) e a largura (vw) da viewport.
    • Exemplo:
      • Em uma tela de 800px de largura e 600px de altura:
        • 1vmin = 6px (1% da menos dimensão.)
        • 1vmax = 8px (1% da maior dimensão).

  6. ch

    • Baseia-se na largura média do caractere 0 na fonte aplicada ao elemento.
    • Útil para largura de texto e caixas.

  7. ex

    • Baseia-se na altura da letra "x" minúscula na fonte usada.
    • Menos comum devido à inconsistência entre fontes.

Benefícios das Medidas Relativas

  • Flexibilidade:Adaptam-se ao contexto do elemento e ao ambiente (dispositivos ou tamanhos de tela).
  • Responsividade: Facilitam a criação de layouts que se ajustam a diferentes resoluções.
  • Consistência: rem e % ajudam a manter proporções consistentes em todo o design.

Exemplo Comparativo



html {
    font-size: 16px; /* Define base para rem */
{

.container {
    widht: 50%; /* 50% da largura do elemento pai */
    padding: 2em; /* Baseado no tamanho da fonte do elemento pai */
    margin: 1rem; /*Baseado no tamanho da fonte do elemento raiz */
    font-size: 1.5rem;/* 24px (16px x 1.5) */
}

.fullscreen {
    widht: 100vw; /*100% da largura da viewport */
    height: 100vh; /* 100% da altura da viewport */

Quando Usar Medidas Relativas?

  • em: Para espaçamentos dependentes do tamanho da fonte de um elemento.
  • rem: Para tamanhos consistentes baseados no tamanho raiz.
  • %: Para larguras e alturas relativas ao contêiner.
  • vh e vw: Para ajustar elementos ao tamanho da tela (como cabeçalhos ou banners em tela cheia).

Medidas relativas são fundamentias para designs fluidos e responsivos!
